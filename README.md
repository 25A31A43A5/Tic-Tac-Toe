# Tic Tac Toe (C++)

A simple console-based Tic Tac Toe game written in C++.  
The player uses **X** and the computer uses **O**.

## How to Compile
g++ main.cpp -o tictactoe
./tictactoe
## Features
- Player vs Computer
- Random moves by the computer
- Clean board layout
- Automatic win detection
